\section{Math Example}

A set $\Set$ in $\Rn$ is said to be convex if
  \[
      \x_1, \x_2 \in \Set \imps \y\x_1 + (1 - \y) \x_2
      \in \Set \tx{ for all } 0 < \y < 1
  \]
